# SimpleGames

A small vanilla web project with multiple playable mini-games on a single page.

## Games Included

- Tic-Tac-Toe
  - 2 Players mode
  - Vs Bot mode
  - Bot difficulty: Easy, Medium
- Flappy Bird
  - In-session score tracking
  - In-session high score tracking (resets on page refresh)

## Tech Stack

- HTML
- CSS
- JavaScript (no build tools)

## Run Locally

1. Open `index.html` in your browser.
2. Or serve with any static server from this folder.

## Project Structure

- `index.html` - App layout and game UI.
- `styles.css` - All styles.
- `script.js` - Game logic and interactions.
